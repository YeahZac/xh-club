# 上星河俱乐部 - 设计思考

## 气质与意象

**关键词**: 热烈、促动、星光夜市

场景：夜幕初降的商业街，霓虹红光映着暖橙招牌，人群涌动中手机屏幕亮起一抹渐变红——"今日秒杀，手慢无"。整个视觉策略围绕"促成行动"设计：红橙渐变拉起紧迫感，圆角卡片兜住信任，金色标签钉住优惠信息。没有冷淡的科技蓝，没有克制的极简——这是一台永不停歇的购物引擎。

## 视觉策略

- 摄影/图像方向：商品大图为主，白底+柔和阴影，强调质感
- 图形语言：圆角矩形卡片（14r），渐变色圆形图标入口，红色促销标签

## 配色方案

| 角色 | 色值 | 意象 |
|------|------|------|
| 主色 | #FF2442 | 霓虹红——行动按钮、TabBar 选中态、促销标签 |
| 辅色 | #FF6034 | 暖橙——渐变终点、次要CTA、限时标签 |
| 主渐变 | 135deg #FF2442 → #FF6034 | 首页Banner、促销条、品牌顶栏 |
| 金色 | #FFB800 | 会员/积分/等级标识 |
| 背景色 | #F4F4F4 | 页面底色，让白色卡片浮出 |
| 文字主色 | #1A1A1A | 标题、商品名 |
| 文字辅色 | #888888 | 描述、副标题 |
| 分割线 | #F0F0F0 | 卡片间、列表项间 |

## 字体排版

- 中文：PingFang SC / system-ui
- 标题：16-20px / 700
- 正文：13-14px / 400
- 辅助文字：11-12px / 400
- 价格：18-24px / 800, #FF2442

## 动效与交互

- 按钮点击：0.15s ease 缩放 0.96
- 页面转场：右滑入/左滑出，0.3s ease
- 列表加载：Skeleton 骨架屏
- Tab切换：下划线跟随动画

## 设计禁忌

- 禁止科技蓝/紫渐变——这是电商不是SaaS
- 禁止纯灰按钮——每个CTA都要有红色基因
- 禁止过大留白——信息密度优先
- 禁止细线边框卡片——用阴影代替边框
