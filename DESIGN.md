# 粤商汇 — 设计思考

## 气质与意象

**关键词**: 深邃、笃定、儒商会所

场景：推开一扇深色木门，步入珠江畔的私人商务会所。深墨蓝的墙面衬着暖金壁灯，皮革沙发沉淀出岁月质感，大理石桌面倒映着窗外天际线。空气中有淡淡的檀木与咖啡气息——这里没有浮躁的霓虹，只有穿西装的人低声谈着几个亿的生意。屏幕上的每一处深蓝都在说"沉稳"，每一抹金色都在说"值得信赖"，每一张大图都在说"这是真实的人、真实的项目"。

## 视觉策略

- 摄影/图像方向：商务场景大图为主——城市天际线、握手签约、路演现场、高层建筑，色调偏暖、光影对比强，营造"高端商务"氛围
- 图形语言：大圆角卡片（16r），大面积图片+少量文字叠加，金色线条点缀，深蓝渐变底栏
- 图片风格：暗角虚化、暖色调、低饱和高对比，杜绝扁平插画和卡通图标

## 字体排版

- 中文：PingFang SC / system-ui，标题可用 font-weight:700 营造力度
- 标题：18-22px / 700，搭配金色下划线装饰
- 区块标题：16px / 600，左侧4px金色竖线
- 正文：14px / 400
- 辅助文字：12px / 400，text-gray-500
- 数字/金额：DIN Alternate / tabular-nums，18-28px / 700

## 动效与交互

- 页面转场：右滑入/左滑出，0.3s ease
- 卡片点击：0.12s ease 轻微下沉 (scale 0.98)
- 列表加载：Skeleton 骨架屏，脉动动画
- Tab切换：下划线滑动跟随
- 金色元素悬停：微光闪烁效果（仅H5）

## 设计禁忌

- 禁止红色促销风——这是商会不是电商
- 禁止科技蓝/紫渐变——不要做成SaaS后台
- 禁止纯灰无金色——金色是品牌灵魂
- 禁止小图标堆砌——用大图说话
- 禁止细线1px边框卡片——用阴影营造层次
- 禁止卡通/扁平插画——只用真实商务摄影
- 禁止大面积留白——商会信息密度要够

## 星河平台俱乐部 — 活动邀约报名页

**气质关键词**: 星河璀璨、商务精英、尊享邀约

场景：夜幕降临，城市天际线上的灯火如同星河倒映。推开厚重的玻璃门，步入一个灯光柔和的空间——深蓝底色上点缀着金色星芒，水晶灯折射出低调奢华。每一位来宾都收到烫金请柬，字迹端正有力。这不是一场普通的聚会，而是一次精英的交汇——星河之上，共启未来。

**报名页设计方向**：
- 顶部深蓝渐变Hero区 + 金色标题 + 星芒装饰
- 会议信息卡片：时间、地点、地铁指引，配以金色图标
- 天气提醒条：暖琥珀色调，温馨提示
- 报名表单：白色卡片，金色按钮，优雅的输入体验
- 底部金色渐变分割线

**管理后台设计方向**：
- 深色顶栏 + 白色内容区
- 数据统计卡片（报名人数、行业分布等）
- 表格列表 + 搜索 + 删除 + 导出按钮
- 管理员密码登录弹窗
